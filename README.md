## 📄 Descripción

Proyecto orientado a la gestión integral de una peluquería, enfocado en el control y registro de costos e ingresos del negocio. 

El sistema permite llevar un seguimiento ordenado de los servicios realizados por el staff, facilitando el análisis económico y la toma de decisiones. Desarrollado bajo la arquitectura **MVC (Modelo-Vista-Controlador)**, está pensado como una solución robusta, segura y clara para el manejo interno de una peluquería en un entorno local.

### ✨ Características Principales
* **Autenticación de Roles:** Acceso diferenciado para Administradores (control total) y Staff (panel de trabajo diario).
* **Tablero Diario:** Registro ágil de servicios por profesional con cálculo automático de comisiones (Local/Peluquero).
* **Análisis de Rendimiento:** Gráficos interactivos de ingresos y ranking tipo "Leaderboard" del Peluquero del Mes.
* **Exportación de Datos:** Historial de cajas cerradas exportable a formato PDF.
* **Gestión de Proveedores:** Catálogo de insumos con sistema de carrito de compras para simular pedidos.
* **UX/UI:** Diseño moderno, responsivo y soporte para *Modo Oscuro* persistente.

## 🛠 Tecnologías utilizadas

* **Frontend:** HTML5, CSS3 (CSS Grid, Flexbox, Variables), Vanilla JavaScript (ES6+).
* **Backend:** PHP (PDO para conexiones seguras a la base de datos).
* **Base de Datos:** MySQL (Diseño relacional).
* **Librerías Externas:** * [Chart.js](https://www.chartjs.org/) (Para la visualización de datos estadísticos).
  * [jsPDF](https://parall.ax/products/jspdf) & AutoTable (Para la generación de reportes).
  * [Font Awesome](https://fontawesome.com/) (Iconografía).
